naturf.utils.rasterize_polygon_shapefile(input_shapefile: str, target_cell_size: Union[float, int], save_raster: bool = True, output_raster_directory: Optional[str] = None, target_field: Union[str, List[str]] = 'footprint', default_value: Union[float, int] = 127, no_data_value: int = 255) → xarray.core.dataset.Dataset[source]

Rasterize polygon shapefile to desired resolution.

Parameters

  • input_shapefile (str) – Full path with file name and extension to the input shapefile.

  • target_cell_size (Union[float, int]) – Target cell size of the output raster.

  • save_raster (bool) – True, if writing raster to file. False, if not.

  • output_raster_directory (Union[str, None]) – If save_raster is True, the full path to the directory to write the output raster file to.

  • target_field (str) – Field name to use for rasterization. Default field “footprint” will be used if none specified and written to file using the value of the default_value argument.

  • default_value (Union[float, int]) – Value used to write to the target field if it does not exist in the input shapefile.

  • no_data_value (int) – No data value to use as the fill value for non-populated cells.

Returns

An xarray Dataset object of the rasterized polygon.
